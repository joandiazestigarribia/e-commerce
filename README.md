# E-Commerce Full-Stack

## Objetivo

Este proyecto no busca ser un e-commerce completo con checkout, sino demostrar arquitectura moderna y mejores prácticas de desarrollo.
Incluye: Home, Product Detail Page y Login/Register

## Arquitectura
- frontend/          # React 18 + TypeScript + Vite
- backend/           # NestJS + PostgreSQL + TypeORM

## Características Principales

### Frontend
- **Vite** 
- **Autenticación JWT** 
- **Lazy Loading** 
- **Testing completo** 
- **UI Moderna**

### Backend
- **NestJS** 
- **PostgreSQL**
- **JWT Strategy**
- **Testing** 

### DevOps
- **Docker**
- **CI/CD** 
- **ESLint strict**
